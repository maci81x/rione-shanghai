COME USARE QUESTO PACCHETTO
============================

1. Estrai questo zip DENTRO la cartella del tuo progetto React
   (es. ~/Sites/rione-shanghai/) — i file si posizioneranno da soli
   nei posti giusti perché rispecchiano già la struttura src/.

2. Contiene:
   - src/assets/drago.svg           -> il file SVG originale del drago
   - src/components/common/DragonLogo.jsx  -> componente React pronto,
     usa fill="currentColor" quindi eredita il colore CSS (nero su sfondo
     chiaro, giallo su sfondo nero header/login)
   - public/drago.svg                -> copia per favicon/manifest PWA

3. Poi incolla il prompt fornito in chat a Claude Code per farlo
   collegare (import) questo componente ovunque serve e completare
   l'ottimizzazione mobile.
